Integrates Qody's Framework into your site, giving it that competative, stupidly crazy done-for-you boost of awesome needed by the rest of Qody's plugins.

== Description ==

Qody's Owl Emporium is the one-stop shop for all things useful & automated for web-based businesses. Everything from custom scripts, Wordpress plugins, themes, bug fixing and more.

If it's done manually, chances are Qody's owls can automate it.

For more information about Qody's Owl Emporium or it's various prestigious workers, head on over to http://plugins.qody.co for the latest scoop.

== Installation ==

1. Upload `qodys-framework` to the `/wp-content/plugins/` directory
2. That's it!  There's no need to activate this plugin, as the framework simply has to "exist" on the server to function
